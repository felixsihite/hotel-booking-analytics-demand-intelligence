# Processed Data

This folder is generated by the Python analytics pipeline.

Key local outputs include:

- `hotel_bookings_cleaned.csv`
- `monthly_demand.csv`
- `forecasting_input_monthly.csv`
- `customer_segment_performance.csv`

These full processed files are intentionally kept out of GitHub. The deployed Streamlit app uses the compact package in `streamlit_app/data/`.
